# Library Form
_Library Form_ is a **Library Registration Form** developed as part of the course "[Servicios de Internet](https://secretaria.uvigo.gal/docnet-nuevo/guia_docent/?ensenyament=V05G301V01&assignatura=V05G301V01301&any_academic=2023_24)" in the Telecommunications Engineering Degree at the Universidad de Vigo (2023 - 2024).

## About The Project
This project implements an interactive web page that allows users to register in a library lending system through an HTML-based form. The application integrates core web development concepts such as structured page design with HTML, visual styling using CSS, and client-side validation with JavaScript and regular expressions to ensure correct user input before submission.

The project features:
- Structured HTML layout for a complete registration interface.
- CSS styling for improved usability and visual presentation.
- JavaScript-based validation of form fields using regular expressions.
- Error handling and user feedback for incorrect input data.
